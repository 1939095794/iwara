# iwara本地化

### 项目说明

iwara_spider：

​	爬虫，用于抓取iwara资源，包括数据库信息统计和视频资源下载

iwara_localplay:

​	本地化仿站，搭建nodejs服务器实现本地播放，收藏等功能
